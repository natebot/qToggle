Code

initializing the plugin:

$(function(){
    $('body').qToggle();
});

This will place click event listeners on all elements in the body that have the data attribute 'data-qtoggle-selector'. As a best practice apply the plugin to a limited scope inside the body, such as $('body','.main-content').qToggle();

initializing the plugin with options:

$(function(){
    $('body').qToggle({
        'effect':'slideToggle',
        'targets':'next',
        'eventArgs':{'duration':'fast','easing':'swing'}
    });
});

You may pass options into the plugin to control many aspects its behavior. The plugin accepts one argument of an object. Look in the source code for what options can be passed to the plugin (until more documentation can be provided here). In this example the slideToggle() function will be called instead of the default Toggle() and an DOM element next to the control will be toggled instead of the previous element. The slideToggle will be given 'fast' and 'swing' as arguments to control its animation.

html control and target:

<a href="#" data-qtoggle-selector=".target" data-qtoggle-text="show info" >hide info</a>

<div class="target" >I am the div that you can hide and show.</div>

the <a> is our control element. The <div> is our control's target element. To apply a toggle effect simply add data-qtoggle-selector to your control element and set its value to a jQuery selector string which which will select your desired target element.
